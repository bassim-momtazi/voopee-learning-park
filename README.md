# voopee-learning-park
# پارک یادگیری ووپی (Woopy Learning Park)
## مشخصات کامل محصول و پلتفرم

---

## نمای کلی پروژه

پارک یادگیری ووپی یک اکوسیستم یادگیری بازی‌محور برای کودکان ۴ تا ۹ سال است که به صورت یک پارک یادگیری دیجیتال تعاملی طراحی شده است.

این پلتفرم صدها ابزار آموزشی تعاملی (مینی‌اپلیکیشن‌ها) را ارائه می‌دهد که در دنیاهای موضوعی مختلف سازماندهی شده‌اند؛ جایی که کودکان از طریق بازی یاد می‌گیرند.

این سیستم نه تنها به عنوان یک محصول، بلکه به عنوان یک اکوسیستم پلتفرمی طراحی شده است که به توسعه‌دهندگان شخص ثالث اجازه می‌دهد بر اساس استانداردهای فنی ووپی، ابزارهای آموزشی بسازند.

### ماموریت ووپی
عدالت یادگیری. فراهم آوردن بهترین امکانات یادگیری برای هر شخص در هرکجا و متناسب با خواست و نیار خودش

### فلسفه برند
یاد بگیر → کسب کن → فرصت یادگیری برای دیگران ایجاد کن

---

# 1. مفهوم اصلی محصول

ووپی به شکل یک شهر یادگیری / پارک یادگیری طراحی شده است.

در داخل این پارک چندین قلمرو یادگیری وجود دارد که هر کدام روی یک حوزه مهارتی تمرکز دارند.

هر فعالیت آموزشی یک **ماموریت (Widget)** در داخل این قلمروها است.

کودکان در پارک گردش می‌کنند و ماموریت‌ها را کامل می‌کنند.

# 8. متادیتای ویجت

فیلدها:

- id
- name
- version
- age_range
- category
- languages
- stages
- difficulty
- skills
- duration
- offline

نمونه:

    {
      "id": "math_addition_train",
      "name": "Addition Train",
      "version": "1.0",
      "age_range": "5-7",
      "category": "math",
      "stages": 5,
      "languages": ["en","fa","es"],
      "offline": true,
      "duration": "5min"
    }
